# Tower Defense Case Study

Проект з командним геймплеєм та різними расами персонажів.

## Як запустити гру

1. Відкрийте проект в Unity Editor
2. Перейдіть до сцени Bootstrap: `Assets/Scenes/Bootstrap.unity`
3. Натисніть кнопку **Play** для входу в Play Mode

## Огляд гри

При старті геймплею гравцю випадковим чином обирається:
- **Одна команда** з 4 доступних команд (Синя, Червона, Жовта, Зелена)
- **Одна раса** з 4 доступних рас (Людина, Орк, Ельф, Гном)

## Конфігурація

### Налаштування персонажів

Конфігурації рас персонажів знаходяться в:
```
Assets/ScriptableObject/Characters/
```

Доступні конфігурації персонажів:
- `Config_Character_Human.asset`
- `Config_Character_Orc.asset`
- `Config_Character_Elf.asset`
- `Config_Character_Dwarf.asset`

Для кожної раси персонажів можна налаштувати наступні параметри:
- **Швидкість руху** (`_moveSpeed`) - Швидкість пересування персонажа
- **Швидкість атаки** (`_attackSpeed`) - Час між атаками
- **Сила атаки** (`_attackDamage`) - Кількість пошкоджень за атаку
- **Радіус атаки** (`_attackRadius`) - Радіус виявлення атаки
- **Здоров'я** (`_healthPoint`) - Очки здоров'я персонажа

### Налаштування команд

Конфігурації команд знаходяться в:
```
Assets/ScriptableObject/Teams/
```

Доступні конфігурації команд:
- `Identifier_Team_Blue.asset`
- `Identifier_Team_Red.asset`
- `Identifier_Team_Yellow.asset`
- `Identifier_Team_Green.asset`

Для кожної команди можна налаштувати:
- **Назва** (`_title`) - Назва команди
- **Колір** (`_color`) - Колір команди (значення RGBA)

### Налаштування геймплею

Основні налаштування геймплею знаходяться в:
```
Assets/ScriptableObject/GameplayConfig.asset
```

Ви можете налаштувати:
- **Кількість персонажів в команді** (`_teamCharactersAmount`) - Кількість гравців на команду
  - Значення за замовчуванням: **5**

Цей конфіг також містить посилання на всі активні ідентифікатори команд та конфігурації персонажів, які використовуються в грі.

## Структура проекту

```
Assets/
├── Scenes/
│   └── Bootstrap.unity          # Головна стартова сцена
├── ScriptableObject/
│   ├── Characters/              # Конфігурації рас персонажів
│   ├── Teams/                   # Конфігурації команд
│   └── GameplayConfig.asset     # Основні налаштування геймплею
└── Scripts/                     # Скрипти гри
```
